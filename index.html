User guide

What this app does

AC App is a local-first PWA for 2nd camera assistants. It stores projects, production days, and take notes directly on your device with autosave.

How to use it

  1. Create a new project from the home screen.
  2. Add production days inside the project.
  3. Add takes quickly, update notes, and move on. Everything autosaves.
  4. Export a PDF report or save a project JSON backup any time.

Backup and restore

Save to device exports one project as JSON. Backup all exports all projects in one file. Use Restore backup from the home screen to import previously saved data.

Install on phone

Open the app in Safari or Chrome, then use Add to Home Screen. It will launch in app style once installed.

Important note

Data is stored on your device. Clearing browser storage or uninstalling the PWA may remove local data, so keep regular backups.
