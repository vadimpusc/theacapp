# AC App

A local-first PWA for 2nd camera assistants.

## Features
- Autosave with IndexedDB
- Add, duplicate, delete projects
- Project-level production metadata
- Optional B Unit fields
- Add production days and takes
- Save one project as JSON
- Backup all projects as JSON
- Restore from backup JSON
- Export a simple PDF report
- Light and dark mode
- Offline-capable PWA with service worker

## Host on GitHub Pages
Upload the contents of this folder to a GitHub Pages repository root.

## Install on mobile
Open the site in Safari or Chrome and use Add to Home Screen.

## Notes
- Data is stored on the device, so keep regular backups.
- Reset All permanently wipes local data.
